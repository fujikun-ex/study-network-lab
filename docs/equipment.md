# 購入機材リスト

予算目安: 〜10万円。中古はYahoo!オークション・メルカリ・ハードオフが主な入手先。
価格はおおよその相場であり、時期・状態によって変動する。

## Phase 1〜2 向け（必須構成）

### L2スイッチ: Cisco Catalyst 2960S-24TS-L × 2台

| | 価格目安 |
|--|---------|
| 中古 | 3,000〜8,000円/台 |
| 新品 | EOL（販売終了）のため実質中古のみ |

- GigE 24ポート、VLAN・STP・トランク設定が一通りできる
- Cisco IOS の CLI 操作に慣れるのに最適
- 2台あるとスイッチ間トランクやSTPの検証ができる
- `2960X-24TS-L` は後継で中古 8,000〜20,000円、IOSが新しく長く使える

## 購入時の注意点

- **Cisco機器はIOSのバージョンを確認する** — 古すぎるとコマンドが異なる場合がある
- **ファンの騒音に注意** — Catalyst 2960S・3560 はファンがうるさい。自室で使う場合は `show environment` でファン状態を確認
- **電源ケーブル付属を確認する** — 中古はケーブルなしの場合がある
- **FortiGateのシリアルをメモする** — ライセンス状態の確認に使う
