# ネットワーク学習 大枠計画

## 学習の方向性

概念理解 → ネットワーク深掘り → セキュリティ の順で段階的に進める。
実機を使って手を動かしながら、「なぜそう動くのか」を体感ベースで理解することを重視する。

---

## Phase 1: 概念理解

**ゴール**: L2〜L4 の基本的な仕組みを説明できる。「なぜ繋がるのか」「なぜ届くのか」がわかる。

| トピック | 内容 |
|---------|------|
| L2 スイッチング | MACアドレス学習、フレーム転送、VLANによるネットワーク分離 |
| L3 ルーティング | IPアドレス、サブネット、静的ルーティング、DHCP |
| L4 TCP/UDP | コネクション確立、ポート番号、信頼性制御の違い |

---

## Phase 2: ネットワーク深掘り

**ゴール**: 障害を自分で調査・切り分けできる。冗長構成や動的ルーティングの考え方がわかる。

| トピック | 内容 |
|---------|------|
| STP | ループ防止、ポートの状態遷移 |
| 動的ルーティング | OSPF の基礎、ルーティングテーブルの変化を観察 |
| NAT | アドレス変換の仕組み、自宅ルーターとの関係 |
| トラブルシューティング | パケットキャプチャ、切り分け手順、ログの読み方 |
| 冗長構成 | VRRP、リンクアグリゲーションの基礎 |

---

## Phase 3: セキュリティ

**ゴール**: 「守る側」の視点でネットワークを設計・評価できる。

| トピック | 内容 |
|---------|------|
| ACL / Firewall | パケットフィルタリング、ステートフルインスペクション |
| VPN | IPsec、トンネルの仕組み |
| 攻撃手法の基礎 | ポートスキャン、ARP スプーフィング、DoS の仕組み |
| セキュリティ設計 | セグメント分離、DMZ、最小権限の考え方 |

---

## 機材・環境

- 実機（スイッチ・ルーター）: 予算〜10万円
- 補助: Raspberry Pi（サーバー・クライアント役）、Wireshark、GNS3/EVE-NG（仮想環境）
