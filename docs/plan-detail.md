## Phase 1: 概念理解

### L2: スイッチングと VLAN

#### 目標
- スイッチがどのようにフレームを転送するか説明できる
- VLANでネットワークを論理分離できる
- タグVLAN（トランク）とアクセスポートの違いがわかる

#### 検証テーマ

**① MACアドレス学習を確認する**
- PC を接続して `show mac address-table` を確認する
- MACアドレスがどのポートに学習されるかを観察する
- わかること: スイッチがフラッディングからユニキャスト転送に切り替わる仕組み

**② VLAN を設定してネットワークを分離する**
- VLAN 10 / VLAN 20 を作成してポートを割り当てる
- 同VLAN間は通信できて、異VLAN間は通信できないことを確認する
- わかること: 物理的に同じスイッチでも論理的に分離できる

**③ トランクポートを設定する**
- スイッチ間をトランク接続して複数VLANを通す
- `show interfaces trunk` で確認する
- わかること: 1本のケーブルで複数VLANのトラフィックを運ぶ仕組み

---

### L3: ルーティングと DHCP

#### 目標
- サブネットを設計してIPアドレスを割り当てられる
- 静的ルートで異なるサブネット間を通信させられる
- DHCPサーバーを構築してIPを自動配布できる

#### 検証テーマ

**① 静的ルーティングで異なるサブネットを繋ぐ**
- 192.168.1.0/24 と 192.168.2.0/24 をルーターで接続する
- `ping` / `traceroute` で経路を確認する
- わかること: デフォルトゲートウェイがなぜ必要か

**② ルーティングテーブルを読む**
- `show ip route` を確認して経路の種別（C/S/O等）を理解する
- わかること: パケットがどの経路を選ぶか判断する仕組み

**③ DHCPサーバーを構築する**
- ルーター or Raspberry Pi で DHCP サーバーを立てる
- クライアントが自動でIPを取得することを確認する
- わかること: DISCOVER → OFFER → REQUEST → ACK の流れ

---

### L4: TCP / UDP

#### 目標
- TCP の3ウェイハンドシェイクを説明できる
- TCP と UDP の用途の違いがわかる
- ポート番号の役割がわかる

#### 検証テーマ

**① 3ウェイハンドシェイクをキャプチャする**
- Wireshark で SYN → SYN-ACK → ACK を確認する
- わかること: TCPがコネクションを確立する手順

**② TCP と UDP の挙動の違いを観察する**
- `iperf3` で TCP/UDP それぞれのスループットを計測する
- パケットロス時の挙動の違いを確認する
- わかること: 信頼性制御のコストと UDP の使いどころ

**③ ポート番号とアプリケーションの対応を確認する**
- `ss -tlnp` / `netstat` で待ち受けポートを確認する
- `nmap` でポートスキャンして開いているサービスを確認する
- わかること: ポート番号がどのようにアプリを特定するか

## Phase 2: ネットワーク深掘り

> Phase 1 完了後に詳細化予定

### STP（スパニングツリー）
- ループが発生する構成を作って STP の動作を観察する
- ルートブリッジの選出過程を確認する
- ポートの状態遷移（Blocking → Listening → Learning → Forwarding）を追う

### 動的ルーティング（OSPF）
- 複数ルーターで OSPF を設定して経路を自動交換させる
- リンク障害時にルーティングテーブルが切り替わることを確認する

### NAT
- プライベートIPからインターネットへのアクセスをキャプチャして変換前後を比較する
- NAPT（PAT）で複数端末が1つのグローバルIPを共有する仕組みを確認する

### トラブルシューティング
- 意図的に障害を起こして切り分け手順を練習する
- `ping` → `traceroute` → `show interface` → `show log` の順で調査する

## Phase 3: セキュリティ

> Phase 2 完了後に詳細化予定

### ACL / Firewall
- スイッチ・ルーターに ACL を設定して通信を制御する
- ステートフルファイアウォールと ACL の違いを理解する

### VPN
- IPsec トンネルを構築してサイト間 VPN を検証する

### 攻撃手法の基礎（理解目的）
- ARP スプーフィングの仕組みと検知方法を理解する
- ポートスキャンがどのように見えるかをログで確認する

### セキュリティ設計
- DMZ を含むネットワーク構成図を設計する
- セグメント分離の考え方を実機構成で試す
