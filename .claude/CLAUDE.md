# CLAUDE.md

## このディレクトリの目的
このディレクトリは、ネットワーク学習用の検証環境と学習メモを管理するためのものです。
実際に機器を触りながら、ネットワークの基礎概念・設定例・検証結果を整理します。

## ゴール
- L2 / L3 / VLAN / DHCP / NAT / Firewall などの基本概念を理解する
- 実機または検証環境で設定を試す
- 試した内容を Markdown で記録する
- 将来的に再現できる形で構成図・設定例・検証結果を残す

## このリポジトリで管理するもの
- 学習計画
- 学習メモ
- 構成図
- 機器ごとの設定メモ
- 検証ログ
- 詰まったポイントと学び
## Claude への依頼方針
Claude は以下を支援する:
- 学習計画の整理
- 実施手順の文章化
- ネットワーク概念の説明
- 検証メモの構造化
- Markdown テンプレート作成
- 構成図のたたき台作成
- 実機学習の進め方提案

Claude は以下を避ける:
- 根拠なく設定値を断定すること
- 秘密情報の埋め込み
- 実環境への危険な変更を前提にした提案
